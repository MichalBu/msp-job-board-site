2. What we offer

RAGDiag is a self-serve software subscription that diagnoses failures in retrieval-augmented generation (RAG) pipelines. For failure cases you provide, it isolates which pipeline component — document chunking, embedding, retrieval ranking, reranking, or query preprocessing — is responsible, and ranks the highest-impact fix. The Starter plan is a recurring monthly subscription of $99/month covering up to 500 failure cases diagnosed per month. RAGDiag is currently in early access. The optional pre-purchase "free audit" is a one-time diagnostic we run manually as an introduction; it is provided free of charge and is not a paid service.
